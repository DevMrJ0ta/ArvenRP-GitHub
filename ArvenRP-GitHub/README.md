# Historia do ArvenR

Sejam todos bem-vindos à cidade de **Arven RP**, um espaço criado para proporcionar uma experiência equilibrada entre **50% PvP e 50% RP**, séria, divertida e imersiva para toda a comunidade.

Este projeto foi fundado e é gerido pelos proprietários **Larinha** e **Biggie**, contando ainda com o apoio dos cofundadores **Silva** e **João**, que trabalham diariamente para garantir o melhor ambiente possível a todos os jogadores.

Antes de iniciares a tua aventura em **Arven RP**, é fundamental que leias e compreendas todas as regras apresentadas. O objetivo destas regras é assegurar o respeito entre todos os membros da comunidade, promover um Roleplay de qualidade e garantir uma experiência justa e equilibrada para todos.

{% hint style="danger" %}
📖 **A leitura das regras é obrigatória.**
{% endhint %}

🏢 Caso pretendas criar uma organização, dirige-te à secção **Legal** ou **Ilegal** e segue as instruções disponíveis.

Ao entrares no servidor, assumes o compromisso de cumprir todas as regras estabelecidas. O seu incumprimento poderá resultar em sanções aplicadas pela equipa de staff.

Desejamos-te uma excelente estadia e que vivas muitas histórias inesquecíveis na cidade de **Arven RP**.

❤️ Obrigado por fazeres parte desta comunidade. Esta é a nossa casa, mas também passa a ser a tua.
